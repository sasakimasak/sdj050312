table태그

-행과 열 (그리드 형식 = 표구조 )의 구조를 제공해주는 기능 - 특수 블럭레벨의 요소임
- 위아래 여백이 없으며, 임대영역이 부모태그의 100%, 가용영역은 컨텐츠너비 만큼이다
- 자식, 자손 요소로는 thead, tbody, tr, caption, th, td태그등이 올 수 있다

thead 태그 tr태그

-thead : 머릿말에 해당되는 태그를 묶어주는 기능 -tr : 한 행을 나타내는 기능 -특수 블럭레벨 요소임 -위 아래 여백이 없으며 , 부모태그의 100%를 인식한다
-margin은 적용되나 표현되질 못하며, padding은 적용할 수 없다

caption 태그

-표제목을 기술할 때 사용하는 태그
-table태그안에 맨 처음 기술 해주는것이 좋다.
(맨 마지막에 기술해도 알아서 맨위로, 센터정렬로 온다)
-특수 블럭레벨요소로, 위아래 여백이 없다
-table 태그가 아닌 다른 곳에 기술 되면 caption태그는 적용되지 않는다. 사라짐

th(table heading), td(table description) 태그

-th : 표에서 열칸의 속성명을 기술하는 태그임 컬럼명
-td : 표에서 열칸의 값을 기술하는 태그임.
-width, height, padding 값 조절 가능 margin불가능
블럭레벨 요소를 담을 수 있는 요소임
